## CRUD
[CRUD](https://highload.today/uk/shho-take-crud-prostimi-slovami-funktsiyi-perevagi-ta-prikladi/) — це скорочення від англійських слів, що позначають чотири операції:

Create (створювати);
Read (читати);
Update (оновлювати);
Delete (видаляти).
